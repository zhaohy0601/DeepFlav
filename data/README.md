# Data Directory

This directory is intended for supplementary data files, sample datasets, or data schema documentation.

The full production dataset (gene annotations, search indices, pathway data) is served through the live platform:

> https://cbi.gxu.edu.cn/KPGF/html/#/

Due to size constraints, the following data directories are excluded from this repository:

- `frontend/public/fla-data/` - Flavonoid gene data files (~4.3 GB)
- `frontend/public/web_data/` - Search index and data chunks (~965 MB)
- `frontend/public/gene-index.json` - Gene index file (~9.7 GB)

For bulk data access or collaboration inquiries, please contact the authors via the platform's contact page.
